# Kuruluş ve Yükseliş Padişahları

Bu proje, verilen RTF dosyasındaki Osmanlı kuruluş-yükseliş dönemi bilgilerini kart eşleştirme oyununa dönüştürür.

## Kullanım

`index.html` dosyasını tarayıcıda açın. `Yeni bilgi` düğmesiyle kart çekin, doğru padişah/dönem sütununu seçin, tüm kartlar bitince `Kontrol et` düğmesine basın.
