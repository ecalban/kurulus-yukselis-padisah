# Kuruluş ve Yükseliş Padişahları

Bu proje, verilen RTF dosyasındaki Osmanlı kuruluş-yükseliş dönemi bilgilerini kart eşleştirme oyununa dönüştürür.

## Kullanım

`index.html` dosyasını tarayıcıda açın. `Yeni bilgi` düğmesiyle kart çekin ve padişah/dönem sütununu seçin; kart yerleştiği anda doğru ya da yanlış olduğu gösterilir. Tüm kartlar bitince `Kontrol et` düğmesiyle genel sonucu görebilirsiniz. Zorlanınca `İpucu` düğmesiyle yalnızca seçili karta ait kısa yönlendirmeyi görebilirsiniz.
